About

A simple set of tools for Dungeons & Dragons, geared toward the 5th edition ruleset where applicable.

Code/Issues

The code for this project is freely available under the MIT licence over at trwolfe13/dnd-5e-tools over at GitHub. Feel free to check it out, have a play and maybe even contribute! Or, if you find any problems, feel free to log an issue over at the GitHub repo.

Acknowledgements

This project would not have been possible without input from the below sources.

Wizards of the Coast

All backgrounds, classes, races, spells, etc. are the property of Wizards of the Coast. The Character Generator and similar tools just randomly generate references to these materials.

FantasyNameGenerators

A lot of the names (specifically surnames of the form '[Adjective][noun]', like "Brokenshield") are inspired by fantasynamegenerators.com, and you should definitely check out the site if you get the chance.

Markov Generation

The name generation is largely achieved using Markov models, with my own implemention inspired by otac0n/markov and chriscore/MarkovSharp.
